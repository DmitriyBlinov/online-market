Тестовое задание на вакансию Java Developer (Middle)

Описание:
Реализовать функционал онлайн-магазина, состоящего из: страницы авторизации, страницы покупки и страницы для администрирования таблицы товаров.

Важно: магазин должен быть реализован без использования Spring.

Технологии:
- Java 8
- PostgreSQL
- Hibernate
- Apache Tomcat
- Apache Wicket
- Log4j

Основные требования:
Приложение должно быть устойчиво к нагрузкам от 10К запросов в минуту
Предусмотреть конкурентный доступ к переменным, отвечающим за количество продукта
Тестовое задание предполагает составление WBS, которое будет передано вместе с выполненным тестовым заданием
